# Экосистема Artur King / KINGMODE

Единая карта: сайт, канал, бот, личка.

## Роли

| Точка | Роль | Аудитория |
|-------|------|-----------|
| **arturkingfitness.com** | Бренд, доверие, форматы, заявка, бесплатные инструменты | Холодный / тёплый трафик |
| **@kingmode_fit** | Контент, философия, воронка на сайт и личку | Подписчики канала |
| **@Raschettbot** | Ежедневный лог: тренировки, питание, отчёты | Пользователи бота |
| **@arturking10** | Личный контакт, заявки, сопровождение | Готовые к работе |

## Слоганы (единые)

- **Сайт:** «Вы покупаете не тренировку. Вы строите систему.»

- **Метод:** «Данные вместо ощущений. Система вместо мотивации.»

## Бесплатно на сайте

- `/tools/nutrition` — калькулятор КБЖУ
- `/tools/training-plan` — подбор структуры тренировок
- `/book` — книга «Система вместо мотивации» (12 глав)

## Воронка

```
Канал (пост + кнопки)
    ├─→ Сайт (форматы / инструменты / книга)
    ├─→ Личка @arturking10 (заявка, вопрос)
    └─→ Бот @Raschettbot (дневник, удержание)

Сайт
    ├─→ /contacts — заявка
    ├─→ Telegram @arturking10
    └─→ Бот (из постов / CTA)
```

## Кнопки под постом канала

1. **Сайт** → `https://arturkingfitness.com`
2. **Личка** → `t.me/arturking10`
3. **Бот** → `t.me/Raschettbot?start=kingmode`

Ссылки только в кнопках — не дублировать в тексте поста.

## Код (источник правды)

- `src/channel/site.ts` — URL, паки, пути
- `src/channel/brand.ts` — кнопки, описания, ссылки
- `src/channel/posts.ts` — очередь постов канала
- `docs/site-corpus.json` — снимок сайта (краул)

## Env (Railway / .env)

```
KINGMODE_SITE_URL=https://arturkingfitness.com
KINGMODE_DM_USERNAME=arturking10
BOT_USERNAME=Raschettbot
TELEGRAM_CHANNEL_ID=@kingmode_fit
CHANNEL_POST_ENABLED=1
```

## Контент-цикл канала

Посты чередуют темы: система → форматы → данные → дисциплина → бесплатные инструменты → сопровождение → CTA. Каждый пост ведёт на кнопки (не дублирует ссылки в тексте).
