## Манера речи (паттерны)

### Структура
1. **Зацепка** — личное или наблюдение («Раньше я тоже…», «Сейчас понимаю…»)
2. **Разворот** — «Но по факту» / «Хотя по факту»
3. **Простой вывод** — одна фраза, без воды
4. **CTA** — коротко: «Забирай.» (не длинный блок ссылок в теле поста)

### Предложения
- Короткие. Часто **одна мысль = одна строка**
- Много **пустых строк** между блоками
- Списки через **«—»**, не через 1️⃣2️⃣ и не длинные `<b>`-заголовки
- Разговорные связки: «вот появится настроение», «по факту», «реально работает в жизни»

### Лексика
- Простые слова. **Без** RIR, DUP, периодизация, MEV в канале
- Анти-мотивация: «бесполезной мотивации», «мотивационного шума», «без фальши»
- Анти-усложнение: «секретные схемы», «идеальные условия»
- База: режим, тренировки, питание, сон, регулярность
- Английское точечно: lifestyle, Phuket (не перегружать)

### Тон
- От первого лица, опыт практика
- Спокойно, уверенно, **без крика и эмодзи-спама**
- Не учит как учебник — **говорит как человек**
- Эмодзи: редко, по делу (📖 в гайде, 🇹🇭 в локации)

### Запрещено в автопостах (не звучит как Артур)
- «Прогрессивная перегрузка — не миф»
- Списки из 5 пунктов с ▪️ и жирным HTML
- «KINGMODE — система из трёх вещей» с нумерацией
- Длинный CTA внутри текста (ссылки — кнопки под постом)
- Канцелярит: «оптимум», «неснимаемый стандарт», «алгоритм»

## Шаблон поста

```
[Личное / наблюдение — 1–2 строки]

[Короткое развитие мысли — 2–4 строки]

[Разворот: Но по факту / Хотя по факту]

[Вывод — 1 строка]

[Опционально список через —]
— пункт
— пункт

[Финал — удар одной фразой]
```

Длина: **400–900 знаков** (не 1500+).
