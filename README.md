# STRENGTH LAB — личный тренировочный Telegram-бот

Работает 24/7 на Railway (репозиторий → авто-деплой при `git push`).

## Функции

| Кнопка | Что делает |
|---|---|
| ✍️ Свободный ввод | Пишешь в чат `присед 100 5х5` или `жим 60х8, 80х5, 100х3` (разные веса, несколько упражнений с новой строки) — бот парсит и сохраняет. Кнопка «↩️ Отменить» под каждой записью |
| 🎙 Голосовые | Наговорил «присед 100 килограмм 5 подходов по 5» → Whisper (Groq, бесплатно) → тот же парсер. Нужен `GROQ_API_KEY` |
| 📝 Записать тренировку | Пошаговый режим кнопками, если удобнее. Автоматический детект рекордов (вес и расчётный 1RM) |
| 📊 Прогресс | История, рекорд, динамика + график, календарь месяца (🟩 = тренировка) |
| ⏱ Таймер отдыха | Кнопка под каждой записью — напомнит через 2/3/4 минуты |
| 📋 Программа | Планы по неделям: DUP, линейная, волновая, 5/3/1 Wendler, GZCLP. 1RM вводится по каждому движению. Кнопка «❓ Как читать» — глоссарий |
| 🧮 1RM калькулятор | `100×5` → расчёт (Epley+Brzycki+Lander), либо одно число → таблица % от известного максимума |
| ⚖️ Вес тела | Лог веса + график динамики |
| 📈 Отчёт недели | Тоннаж, тренды, застой, баланс жим/тяга. Авто-рассылка по воскресеньям 18:00 (Бангкок) |

## Локальный запуск

1. Токен от [@BotFather](https://t.me/BotFather) → в `.env`: `BOT_TOKEN=...`
2. `npm install`
3. `npm run dev`

## Деплой (Railway)

- Пуш в `main` → Railway пересобирает автоматически (nixpacks: `npm ci` → `npm run build` → `node dist/index.js`)
- Переменные окружения: `BOT_TOKEN`, опционально `GROQ_API_KEY` (голосовые; бесплатный ключ — [console.groq.com](https://console.groq.com))
- Данные в `data.json` (путь через `DATA_PATH`, на Railway смонтируй Volume в `/data`)
- Данные разделены по пользователям (`userId` в каждой записи) — бота можно давать другим, дневники не перемешаются. Старые записи без `userId` при первом запуске привязываются к первому зарегистрированному пользователю

## Структура

```
src/
  index.ts            — бот: меню, хэндлеры, cron-сводка
  db.ts               — JSON-хранилище: тренировки, программы, вес, юзеры
  parser.ts           — парсер свободного текста тренировки
  voice.ts            — распознавание голосовых (Whisper via Groq)
  analysis.ts         — недельный отчёт (rule-based)
  chart.ts            — графики через QuickChart
  calc/orm.ts         — 1RM (3 формулы) + таблица %
  calc/periodization.ts — DUP / линейная / волновая
  calc/templates.ts   — 5/3/1 Wendler, GZCLP
```
