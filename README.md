# STRENGTH LAB — личный тренировочный Telegram-бот

Работает 24/7 на Railway (репозиторий → авто-деплой при `git push`).

## Функции

| Кнопка | Что делает |
|---|---|
| 📝 Записать тренировку | Упражнение → `4×5×120` → сохранено. Автоматический детект рекордов (вес и расчётный 1RM) |
| 📊 Прогресс | История, рекорд, динамика + график |
| 📋 Программа | Планы по неделям: DUP, линейная, волновая, 5/3/1 Wendler, GZCLP. 1RM вводится по каждому движению. Кнопка «❓ Как читать» — глоссарий |
| 🧮 1RM калькулятор | `100×5` → расчёт (Epley+Brzycki+Lander), либо одно число → таблица % от известного максимума |
| ⚖️ Вес тела | Лог веса + график динамики |
| 📈 Отчёт недели | Тоннаж, тренды, застой, баланс жим/тяга. Авто-рассылка по воскресеньям 18:00 (Бангкок) |

## Локальный запуск

1. Токен от [@BotFather](https://t.me/BotFather) → в `.env`: `BOT_TOKEN=...`
2. `npm install`
3. `npm run dev`

## Деплой (Railway)

- Пуш в `main` → Railway пересобирает автоматически (nixpacks: `npm ci` → `npm run build` → `node dist/index.js`)
- Переменная окружения: `BOT_TOKEN`
- Данные в `data.json` (путь через `DATA_PATH`, на Railway смонтируй Volume в `/data`)

## Структура

```
src/
  index.ts            — бот: меню, хэндлеры, cron-сводка
  db.ts               — JSON-хранилище: тренировки, программы, вес, юзеры
  analysis.ts         — недельный отчёт (rule-based)
  chart.ts            — графики через QuickChart
  calc/orm.ts         — 1RM (3 формулы) + таблица %
  calc/periodization.ts — DUP / линейная / волновая
  calc/templates.ts   — 5/3/1 Wendler, GZCLP
```
