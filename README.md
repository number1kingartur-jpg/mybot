# MyBot — личный тренировочный Telegram-бот

## Запуск

1. Создай бота у [@BotFather](https://t.me/BotFather) → скопируй токен
2. Вставь токен в `.env`:
   ```
   BOT_TOKEN=123456:ABC-your-token
   ```
3. Запусти:
   ```
   npm run dev
   ```

## Функции

| Кнопка | Что делает |
|---|---|
| 📝 Записать тренировку | Выбери упражнение → введи подходы×повторения×вес → сохранено |
| 📊 Прогресс | Выбери упражнение → история + график весов |
| 📋 Программа | Активная программа + текущая тренировка; создать новую если нет |
| 🧮 1RM калькулятор | Введи вес×повторения → получи 1RM и таблицу % |

## Данные

Все данные хранятся в `data.json` рядом с ботом. Бэкап = скопировать файл.
